
# Data Engineering Training
### Hexaware Technologies | Vaishali Sashikumar

---

## About This Repository

This repository contains all exercises, assignments, practice notebooks, output visuals, and training documents prepared during the **Hexaware Data Engineering Training Program**.

Work is organized **date-wise** — each folder represents one training session.

---

## Syllabus & Topics Covered

| Module | Topics |
|---|---|
| **SQL & Data Warehousing** | SQL Queries, Joins, Aggregations, Window Functions, Subqueries, DWH Concepts |
| **Python for Data Engineering** | Core Python, File Handling, OOP, Exception Handling |
| **PySpark** | DataFrames, Filtering, Aggregations, Joins, GroupBy, Window Functions, RDD, Spark SQL |
| **Apache Spark** | Spark Architecture, Transformations, Actions |
| **Azure Databricks** | Databricks Notebooks, Delta Lake, DataFrame Operations, SQL Practice Lab |
| **Apache Airflow** | Workflow Orchestration, DAGs, Task Scheduling |
| **Azure DevOps** | CI/CD Pipelines, Version Control, Deployment |
| **NoSQL / MongoDB** | Collections, CRUD Operations, Capstone Projects |
| **Coding Evaluations** | Topic-wise assessments conducted throughout the program |
